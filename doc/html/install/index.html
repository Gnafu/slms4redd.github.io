Installing NFMS in ProductionÂ¶

This module describes the step by step installation and configuration of the nfms platform, as needed in production environments.

The NFMS platform is java-based and multiplatform, so it can be run on top of different operating systems. Nevertheless, Ubuntu or CentOS Linux distributions are recommended.

The installation has been fully tested on Ubuntu 12.10

In this module you will see:

  • Hardware requirements
    • Staging area
    • Dissemination area
  • Software prerequisites
    • JDK 6
    • Tomcat 6
    • Apache 2
    • GDAL
    • PostGIS
  • Preparing multiple tomcat instances
    • Creating the tomcat profiles
    • Check tomcat running instances
    • Make services start at boot time
    • AJP proxying
  • Deploy and configure staging and dissemination GeoServer
    • GeoServer data directory
    • Onlinestats (WPS)
    • Environment settings
  • Deploy and configure staging and dissemination GeoStore
    • Initialize PostgreSQL database
    • GeoStore configuration files
    • Environment settings
  • Deploy and configure staging GeoBatch
    • GeoBatch configuration files
    • Environment settings
  • Deploy and configure Administration interface
    • Application context
    • Environment settings
  • Deploy and configure dissemination Portal
    • Portal configuration directory
    • Portal properties file
    • Environment settings
