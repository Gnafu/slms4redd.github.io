Statistics service data model¶

How do we add statistics? GeoServer doesn’t provide this functionality so we need to have some additional services.

In order to build charts, these sercices need to know what calculations to apply to which layer, and how to present this to the user. This information is modeled in a database called GeoStore.

We’ll see how:

  • Create a layer
  • Add a layer time instance
  • Create statistics
  • Create charts
  • How to add statistics to the portal
