Installing the LiveDVD on the systemÂ¶

In order to increase the system performaces and, in case of live DVD, to have your work persisted for a later session, it is worth to install the LiveDVD on the system. It can be done either by installing the operating system on your local machine or on a Virtual Machine (VM) such as VMWare or VirtualBox.

Note

In order to work smoothly and without issues with GeoServer it is recommended to make the GEOSERVER_DATA_DIR point to a folder on the persistent storage (see next section).

  1. Follow the instruction in the previous section in order to start the LiveDVD.

    Note

    Configuring a brand new VM compatible with Ubuntu, and attaching the LiveDVD to it, also allows you to install the workshop on a virtual environment.

  2. Once you get the LiveDVD boot screen as shown below, type install and press ENTER

    LiveDVD boot screen

  3. Choose your language and click Continue on the Welcome screen.

    LiveDVD install: language selection

  4. Choose your time-zone and click Continue on the Where are you? screen.

    LiveDVD install: time-zone setting

  5. Choose your keyboard layout and click Continue on the Keyboard screen.

    LiveDVD install: keyboard layout

  6. Select the partition where to install the system and click Forward on the Prepare disk space screen.

    Note

    We suggest to not modify the configuration proposed here unless you are an advanced user and know exactly what you are doing.

    LiveDVD install: Installation Type

  7. Select the username and password and the hostname, then click on Forward.

    Warning

    This course envisages the use of a username:unredd with a password: unredd. You can choose any hostname.

    LiveDVD install: Who are you? selection

  8. Check the summary and click Install.

    LiveDVD install: Ready to install summary.

  9. Wait for installation to finish and then restart the system. Remember to remove the LiveDVD from the DVD-ROM before restarting the computer or VM.

    LiveDVD install: installation progress

    LiveDVD install: installation finished

    LiveDVD install: Working system

    Note

    The OS user name and password to use for this workshop are:

    • username: unredd
    • password: unredd

    Note

    For each installed application on the system we used the user unredd with password unredd.

At this point you should have a fully functional environment to run GeoServer, GIS tools and clients on your dataset.
